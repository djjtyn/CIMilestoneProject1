Digital Audio Workstation (DAW)

The purpose of this site is to provide information for people interested in creating their own home recordings and the tools that are needed to enhance the audio quality on the recordings.

A Digital Audio Workstation (DAW) is software used in the recording and post production of audio. They are used in many professional recording studios but can be set up easily enough for use in the comfort of your home.

On this site, you will find the requirements needed to make use of one of these in addition to information on some commonly used tools found within every one of the digital audio workstation programmes available.

There is many different digital audio workstation software available such as Pro Tools, Audacity, Ableton and Audition to name a few. You will find icons on this page for the four software programmes I have named. Each icon is a link, which upon clicking, will take you to a page containing some information on each individual software program and its various features.
Every one of these software programmes have their own strengths and weaknesses but all will work well with recording or improving audio quality for basic home recordings.

Some of the software I have listed is free to download. The ones that cost to download will allow a free trial or allow scaled down version of the full program to be downloaded for free. All the tools mentioned on this website can be found in the free versions of whichever software you choose to download.

There are a number of requirements neccessary to get started with using this software, which I have listed below.

Requirements

Computer

The first requirement for setting up a home recording studio is a computer. A computer is needed to run the software. The software can usually be ran on either desktop computers or laptops. Both Mac and Windows computers can run the software. The higher specification the computer you are working with, the better the software will run on it.

It's important to note that all digital audio workstation software can only run with specific system requirements. The machine specifications required for some DAW systems can be found below:

  • Audacity for Windows
  • Audacity for Mac
  • Ableton for Windows and Mac
  • Pro Tools for Windows and Mac
  • Audition for Windows and Mac

Audio Interface

An audio interface is hardware which you need to get audio into and out from your computer. It consists of inputs and outputs. There are some interfaces with a large number of inputs. The more inputs an audio interface has means the more microphones/instruments you can use to record with at once ie. if theres an audio interface with 8 microphone inputs, the user can record 8 microphones simulataneously.

There are a wide variety of audio interfaces available, all with different prices. Generally speaking, the higher the price of an interface usually reflects a higher quality of recording than the cheaper interfaces. Audio interfaces can be connected to the computer by either USB, firewire or thunderbolt connection types.

You can find a list of some highly rated audio interfaces to help choose which to buy by clicking here.

Microphone

A microphone will be needed if you intend to record vocals or instruments that dont have electric capabilities. However, if you plan on recording only electric instruments, a microphone will not be needed as most electric instruments have line out capibilities that allow a 1/4" jack cable to be inserted and routed direct to the interface.

There are different types of microphones available which are dynamic, condensor and ribbon microphones. It's important to note that if you choose anything other than a dynamic microphone, you will need an interface with phantom power which allows an electrical current to flow to the microphone to turn it on.
