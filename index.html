Digital Audio Workstation (DAW)

A Digital Audio Workstation (DAW) is software used in the recording and post production of audio. They are used in many professional recording studios but can be set up easily enough for use in the comfort of your home.
On this site, you will find the requirements needed to make use of one of these in addition to information on some commonly used tools found within every one of the digital audio workstation programmes available.
There is many different digital audio workstation software available such as Pro Tools, Audacity, Ableton and Audition to name a few. You will find icons on this page for the four software programmes I have named. Each icon is a link, which upon clicking, will take you to a page containing some information for each individual program and various features included in each one.
Every one of these software programmes have their own strengths and weaknesses but all will work well with recording or improving audio quality for basic home recordings.
Some of the software I have listed are free to download whilst some cost to download. The ones that cost to download will allow a free trial or allow scaled down version of the full program to be downloaded for free. All the tools mentioned on this website can be found in the free version of whichever software you choose to download.
There are a number of requirements neccessary to get started with using this software, which I have listed below.

Requirements

Computer

The first requirement in setting up a home recording studio is a computer. A computer is needed to run the software. The software can usually be ran on either desktop computers or laptops. Both Mac and Widnows computers can run the software.
It's important to note that the software can only run with specific system requirements.
The machine specifications required for some DAW systems can be found below:

  • Audacity for Windows
  • Audacity for Mac
  • Ableton for Windows and Mac
  • Pro Tools for Windows and Mac
  • Audition for Windows and Mac

Audio Interface

An audio interface is hardware which you need to get audio into and out from your computer. It consists of inputs and outputs. There are some interfaces with a large number of inputs. The more inputs an audio interface has means the more microphones/instruments you can use to record with at once ie. if theres an audio interface with 8 microphone inputs, the user can record 8 microphones simulataneously.
There are a wide variety of audio interfaces brands all with different price brackets. If the recordings you are going to make are just going to be listened to by yourself and a few others the lower price bracket would be suitable for this, however if you intend to broadcast your recordings to large audience numbers, an interface which is more expensive will be better.

Microphone

A microphone will definatly be needed if you intent to record vocals. However, if you plan on just recording instrumentals with musical instruments the microphone may not be needed, as most amplifiers for electric instruments have line out capibilities that allow a 3/4 jack to be inserted ad routed direct to the interface, There is a wide variety of microphones which are dynamic microphones, condensor microphones and ribbon microphones.
